SCEGLI COME INIZIARE

Cosa vuoi fare?

Puoi creare un nuovo set di flash card oppure aprire un set già scaricato per modificarlo.

GUIDA RAPIDA

Tre passaggi per creare il set di flash card

Apri la guida solo quando ti serve. Puoi iniziare subito anche senza leggerla.

1IniziaScegli se creare un nuovo set di flash card o aprire un progetto già scaricato.
2Crea e controllaInserisci le card una alla volta. In ogni card puoi mettere un’immagine sul fronte, un testo sul retro, il file audio MP3 del testo e uno o più collegamenti esterni. Per le immagini usa solo file PNG. Controlla il risultato con l’anteprima.
3Scegli il risultatoAlla fine scegli il risultato: scarica il progetto completo se vuoi riaprirlo, modificarlo o pubblicarlo online; scarica invece un progetto da pubblicare o consultare se vuoi condividerlo senza permettere modifiche. Dentro “Scarica il progetto” trovi anche le istruzioni e i tutorial per pubblicare su GitHub.

Hai bisogno di altre istruzioni?

Scegli soltanto l’aiuto che ti serve.

Come riaprire un set di flash card

  1. Premi “Apri un set di flash card”.
  2. Seleziona la cartella completa, non il solo file index.html.
  3. Modifica, aggiungi, elimina o riordina le card.
  4. Alla fine premi “Scarica il progetto” per conservare una nuova copia modificabile.

Importante: la cartella deve contenere progetto.flashcards.

PASSO 1

Inizia dal tuo set di flash card

Scrivi il titolo generale. Poi crea le card una alla volta: vedrai subito il risultato.

Come funziona: 1 Aggiungi una flash card. 2 Inserisci il testo del fronte, un’immagine PNG facoltativa, il testo del retro, un audio MP3 e i collegamenti esterni. 3 Trascina le flash card per riordinarle. 4 Controlla i contenuti, i collegamenti e l’ordine delle flash card con l’anteprima.
Formati supportati: immagini esclusivamente PNG (.png) e audio esclusivamente MP3 (.mp3). Gli altri formati vengono bloccati.

Scarica il progetto

Scarica una copia che potrai riaprire e modificare.

Scaricare e continuare a modificare

  1. Scegli una cartella vuota.
  2. Il programma scarica tutti i file del progetto nella cartella scelta.
  3. Conserva la cartella completa senza separare i file.
  4. Per continuare il lavoro, usa “Apri un set di flash card”.

La cartella contiene progetto.flashcards, necessario per riaprire il progetto nell’editor.

Progetto da pubblicare o da consultare

Scarica una copia che potrai condividere senza permettere modifiche. Poi segui il tutorial per pubblicare il tuo set di flash card online e ottenere un link.

Consigliato per la condivisione

Come funziona
  1. Scegli una cartella diversa da quella del progetto modificabile.
  2. Il programma scarica una copia che non può essere modificata con l’editor.
  3. Per consultarla sul computer, apri il file index.html.
  4. Per pubblicarla online e ottenere un link, segui uno dei tutorial qui sotto.

Questo progetto non contiene il file progetto.flashcards.

Pubblicare il set di flash card online

Segui il tutorial che preferisci per pubblicare il tuo set di flash card su GitHub e ottenere un link da condividere.

Pubblicare online con GitHub Pages

GitHub è un sito che permette di conservare e condividere file di progetto. GitHub Pages è il servizio gratuito di GitHub che trasforma i file del progetto in un piccolo sito web con un link da condividere.

Prima scarica il progetto da pubblicare o consultare. Poi:

  1. Accedi a GitHub e crea un nuovo repository pubblico, cioè una cartella online.
  2. Carica nel repository tutti i file e tutte le cartelle del progetto.
  3. Conferma il caricamento con Commit changes.
  4. Apri Settings → Pages.
  5. Scegli Deploy from a branch, poi main e /(root).
  6. Salva. GitHub Pages creerà il link del sito da aprire e condividere.

Importante: non caricare soltanto index.html. Servono tutti i file della cartella.
